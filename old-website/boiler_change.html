Boiler Change

When changing a boiler, I would always recommend using a Valliant boiler as they are efficient, reliable and long lasting.

A Vaillant 624 24kw boiler would cost around £950. These boilers are system boilers and work on a conventional system where you have a boiler and a cylinder for the hot water.

Of course if you want a cheaper boiler, that is fine, and can be done. This would save you up to £150, but it would be unlikely to be as efficient, or reliable. The cheaper option, a 24KW Vaillant Ecotec combi boiler, would cost around £900.
